# 🗂 Portfolio Telegram Bot

Личный бот для хранения и демонстрации портфолио прямо в Telegram.

## ✨ Возможности

- 📁 Добавление проектов с описанием, фото, ссылками и навыками
- 🛠 Отдельный список навыков/технологий
- ✏️ Редактирование и удаление проектов
- 🔗 Быстрое создание текста-портфолио для отправки

---

## 🚀 Запуск за 10 минут (Railway — бесплатно)

### Шаг 1 — Создай бота в Telegram

1. Открой [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Придумай имя и username (напр. `MyPortfolioBot`)
4. Скопируй **токен** — он выглядит так: `7412345678:AAF...`

### Шаг 2 — Загрузи код на GitHub

1. Зарегистрируйся на [github.com](https://github.com)
2. Создай новый репозиторий (New repository → назови `portfolio-bot`)
3. Загрузи файлы: `bot.py`, `storage.py`, `requirements.txt`, `Procfile`
   - Нажми "uploading an existing file" и перетащи все 4 файла
4. Нажми "Commit changes"

### Шаг 3 — Задеплой на Railway

1. Зайди на [railway.app](https://railway.app) → Sign in with GitHub
2. New Project → Deploy from GitHub repo → выбери `portfolio-bot`
3. После создания проекта перейди во вкладку **Variables**
4. Добавь переменную:
   - Key: `BOT_TOKEN`
   - Value: токен из шага 1
5. Перейди в **Settings → Deploy** → убедись что команда `worker: python bot.py`
6. Нажми **Deploy** — через минуту бот запустится!

> 💡 Railway даёт $5 в месяц бесплатно — этого хватит на постоянную работу лёгкого бота.

| Действие | Как |
|----------|-----|
| Открыть меню | `/start` |
| Добавить проект | Кнопка "➕ Добавить проект" |
| Посмотреть проекты | Кнопка "📂 Мои проекты" |
| Добавить навыки | Кнопка "➕ Добавить навык" |
| Поделиться | Кнопка "🔗 Поделиться портфолио" |

---

## 🗃 Хранение данных

Данные сохраняются в файл `portfolio_data.json` прямо на сервере Railway.

> ⚠️ При перезапуске на Railway данные не теряются — файл сохраняется.  
> Для дополнительной надёжности можно подключить Railway Volumes (в настройках).

## 🛠 Локальный запуск (для теста)

```bash
pip install python-telegram-bot==20.7
export BOT_TOKEN="ваш_токен_здесь"
python bot.py
```
